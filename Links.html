Capítulo 1 - O que é HTML


A sigla HTML significa Hyper-Text Markup Language, o que significa Linguagem de marcação de hipertexto.
É a linguagem utilizada para construir a maior parte das páginas EB da Internet.

Capítulo 2 - Conceitos básicos HTML

Uma página Html é, um ficheiro de texto. O texto é composto por tags (etiquetas) e por texto normal.
As tags têm como função marcar e dar efeitos especiais ao texto, para adquirir o formato desehado quando for lido pelo browser.
